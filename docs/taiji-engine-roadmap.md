```
第三层 · 道 — 全智能网络的底层自运行规则
```

> 用户几十年的经验和直觉，变成太极引擎的行为模式，

## 四、核心原则——永不违反

1. **不做模型** — 不训练、不微调、不和模型公司抢饭碗
2. **不锁用户** — 任何模型可用、任何时间可换、数据随时可导出
3. **不碰内容** — 回传只传元数据、对话绝不上传、隐私是底线
4. **过度克制** — 不该做的一律不做，Pi准则兜底
5. **极致简单** — 用户不需要理解"引擎"，只需要感知"好用"
6. **持续进化** — 用了的人越多 → 对所有人越好 → 别人抄不了
