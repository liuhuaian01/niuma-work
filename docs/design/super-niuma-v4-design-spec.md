```
┌────┬─────────┬──────────────────────────┬──────────┐
│    │          │                          │          │
│ Nav│ Chat     │   Main Chat Area         │  Panel   │
│    │ List     │                          │  (toggle)│
│ 64 │ 240      │   flex:1                 │  360min  │
│ px │ px       │   max-width:720px居中     │  px      │
│    │          │                          │          │
└────┴─────────┴──────────────────────────┴──────────┘
 │                                     │
 └──── 左侧区 = Nav + List, 毛玻璃底 ────┘
 └────────── 主区 = 全白底, 内容居中 ──────┘
 └── 面板 = 右侧滑入, backdrop-blur ───────┘
```

### 2.1 布局规则

| 规则 | 值 | 说明 |
|------|-----|------|
| 最小宽度 | `1024px` | 窄于此时隐藏右侧面板 |
| 最大内容宽度 | `720px` | 聊天区居中聚焦 |
| 面板默认 | 关闭 | 点击顶部按钮 → 右侧滑入 |
| 面板宽度 | `280px - 420px` | 可拖拽调整 |
| 毛玻璃区域 | 左侧两列 | `backdrop-filter: blur(20px)` |

### 2.2 暗色模式切换

```css
/* 默认跟随系统 */
@media (prefers-color-scheme: dark) {
  :root { color-scheme: dark; }
  /* 暗色 token 覆盖 */
}

/* 手动切换 */
[data-theme="dark"] { /* 暗色 token */ }
[data-theme="light"] { /* 亮色 token */ }
```

## 十一、动效系统 (Motion System)

### 11.1 缓动函数

```css
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);     /* 弹性 */
--ease-out: cubic-bezier(0, 0, 0.2, 1);                 /* 标准退出 */
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);            /* 标准进出 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.27, 1.55);   /* 弹跳 */
```

### 11.2 动画时长

| 类型 | 时长 | 用途 |
|------|------|------|
| 微交互 | `150ms` | hover变色, 图标旋转 |
| 小过渡 | `250ms` | 面板滑入, 弹窗进出 |
| 页面切换 | `350ms` | 对话切换, 标签切换 |
| 入场动画 | `500ms` | 列表交错入场 (stagger) |

### 11.3 关键动画清单

| 动画 | 触发 | 效果 |
|------|------|------|
| 导航栏 hover | 鼠标悬停 Agent 图标 | 上移 2px + 品牌光晕 |
| 对话卡片 hover | 悬停卡片 | 右移 2px + 背景微变 |
| 消息气泡入场 | 发送/接收新消息 | scale(0.9→1) + opacity(0→1), spring 300ms |
| 打字指示器 | AI 开始回复 | 三个点交替缩放 |
| 面板滑入 | 点击面板按钮 | translateX(100%→0), 300ms ease-out |
| 弹窗入场 | 打开弹窗 | opacity+scale+translateY, 200ms spring |
| 列表交错 | 首次加载/刷新 | 子元素依次 fadeIn+slideUp, 各 50ms delay |
| 按钮按下 | 点击按钮 | scale(1→0.97→1), 150ms |
| Toast 入场 | 操作完成/失败 | translateY(-10→0) + fadeIn, 250ms |
| 发送按钮脉冲 | 输入框有内容 | 渐变背景 + 微发光 animate |

### 11.4 空状态动画 (Loading/Skeleton)

```

│ ████████████████████░░░░ 80% │  ← shimmer bar

```
- `background: linear-gradient(90deg, var(--bg-hover) 25%, transparent 50%, var(--bg-hover) 75%)`
